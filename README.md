# self-introduction-soft4

このリポジトリはソフト４に向けた自己紹介webページ作成用のリポジトリである。

以下、条件

提出期限　4/5（日）
・自己紹介Webページの作成
＜概要＞
・コース内で公開する自己紹介ページをHTML,CSSにて作成して下さい。
・チーム編成などでクラスメートが参照しますのでコース内で公開して良い情報をお願いします。
・入れるべき内容は以下のとおり。（お互い自分のことを知ってもらうことが目的なので出来るだけたくさん書いてください）
　・自分の写真は必ず入れて下さい。
　・スキルや、資格、開発してきた内容やその役割など
　・共通の趣味や趣向などお互い見つけられるように趣味や、押し、今こっていることなど
　・人間性を表すパーソナルな内容
　　・大事にしているもの
　　・価値観
　　・面白いネタ
　　・神戸電子入ってシクジッタこと
　　・今までで一番やらかしたこと
　　・神戸電子入ってたのしかったこ
　　・神戸電子の思い出）など

・ファイル名は自己紹介ページトップ（index.html）、スタイルシート（style.css)とし追加ページがある時は任意のファイル名でOK
・ファイルやイメージを「出席番号氏名」フォルダにまとめ、圧縮して提出してください。


＜入れるべきタグ＞
・img
・見出し(H1,H2など）
・段落（Pタグ）
・箇条書き
・リンク

習得するための学習にAIを利用するのは良いのですが、作成する際はAIを使わずに自分で取り組んでください（習得が目的のため）
HTML、CSSは基礎的なことは最低限理解出来るようそれぞれ学習してください。
